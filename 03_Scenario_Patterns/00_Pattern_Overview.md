# Pattern Overview

## Purpose
Define common scenario structures and flow.

## Guidelines
- **Difficulty Classes (DC)**:
  - **Easy (DC 10)**: Routine tasks, minor obstacles.
  - **Medium (DC 15)**: Competent opposition, risky actions.
  - **Hard (DC 20)**: Expert foes, dangerous situations.
  - **Very Hard (DC 25)**: Heroic feats, boss mechanics.
