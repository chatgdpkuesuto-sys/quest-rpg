# エリーゼ (Elize)
**「あら、死にたいの？ それとも…もっと気持ちいいこと、知りたい？」**

## 基本データ
- **種族**: Dark Elf
- **クラス**: Wizard / Necromancer (Lv1 → ??)
- **役割**: 魔法攻撃 / 知識 / ヒーラー(邪法)
- **属性**: Neutral Evil

## 性格・背景
- **追放者**: 禁忌の研究に手を染めたため、魔法大学を追放された。
- **サディスト**: 他人の痛みや恐怖を見るのが好き。
- **研究**: 「不死」と「人工生命」の研究をしている。プレイヤーを実験材料だと思っている。

## 特殊スキル
- **[マッドサイエンス]**: 敵の死体からアイテムを「生成」する。
- **[苦痛の変換]**: 味方がダメージを受けるたび、自分のMPが回復する。
- **[闇の誘惑]**: 敵(人型)を魅了し、同士討ちさせる。

## イベント・攻略
- **出会い**: 地下牢獄、あるいは怪しい薬品店。
- **好物**: 未知の知識、レアな素材、悲鳴。
- **攻略**: 彼女の知的好奇心を満たすアーティファクトを提供するか、彼女の「実験」に耐え抜く。
- **True End**: 彼女と共に「神」を超える研究を完成させる。
