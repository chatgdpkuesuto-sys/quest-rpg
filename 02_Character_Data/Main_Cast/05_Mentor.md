# 大魔導師 エララ (Master Elara)

**「あらあら、また怪我をして。…本当に手のかかる可愛い子ね」**
**「いい？ 私の愛弟子の髪の毛一本でも傷つけたら…この国ごと消し飛ばすわよ？」**
**「ふふ、300年待ったのよ。…もう、逃がさないわ」**

## 1. パーソナルデータ (Personal Data)

### 基本情報
- **本名**: Elaranel Sylphyria (エララネル・シルフィリア)
- **通称**: エララ師匠、青の魔女 (The Azure Witch)、森の賢者
- **真名**: **[検閲済]** (古代エルフ語で「星を愛する者」)
- **年齢**: 350歳以上 (自称「永遠の17歳と300ヶ月」)
    - **実年齢**: 正確には382歳。この話題を出した者は灰になる。
- **種族**: High Elf (ハイエルフ・始祖の血族)
- **クラス**: Sage (賢者) / Grand Wizard (大魔導師)
- **誕生日**: 1月1日 (元日・全てが始まる日)
- **星座**: 始祖の座
- **血液型**: マナ型 (輸血不可)
- **出身**: 世界樹の森 (World Tree Forest)
- **住居**: 迷いの森の奥にある「魔女の隠れ家」
    - 異次元空間で拡張されており、内装は豪華。

### 外見詳細 (Appearance)
年齢不詳の妖艶な美女。圧倒的な包容力と母性を感じさせるオーラを放つ。
見る者を安心させると同時に、底知れぬ魔力の恐ろしさも感じさせる。

- **身長**: 168cm (ヒール込みで175cm)
    - 少女化時は132cm。
- **体重**: 秘密 (「女性に重さを聞くなんてマナー違反よ？」)
    - 抱き心地は最高に柔らかい。
- **スリーサイズ**: B95(G) / W60 / H92
    - 重力魔法で形を維持しているという噂があるが定かではない。
- **肌の色**: 雪のような白磁の肌 (Porcelain White)。
    - 日光に当たらないため、血管が透けて見えるほど白い。
    - 肌触りはシルクのように滑らかで、常にほんのりと温かい。
    - 香り：ラベンダーと古い書物、そして微かな甘いミルクの香り。
- **髪**:
    - **色**: シルバーブルー (Silver Blue)。光の加減で青白く輝く。
    - **長さ**: 地面に届くほどのスーパーロング。魔法で常にふわりと浮かせているので汚れない。
    - **髪型**: 緩やかなウェーブ。前髪はセンター分けで、知的な額を出している。
    - **手入れ**: 毎晩、召喚した風の精霊に梳かせている。
- **瞳**:
    - **色**: アメジスト (Deep Amethyst)。全てを見透かすような深紫。
    - **特徴**: 感情が高ぶると発光する。魔法行使時は幾何学模様の陣が浮かぶ。
    - **視線**: 常に慈愛に満ちているが、怒るとハイライトが消える。
- **耳**: 長く尖ったエルフ耳。
    - 感情と連動して動く。嬉しいとピーンと立ち、悲しいと垂れる。
    - 弱点でもあり、息を吹きかけられると変な声が出る。

### [Secret Form: 少女化 (Loli Mode)]
魔力が極端に低下した際に発動する防衛本能（幼児退行）。
- **身長**: 132cm
- **体型**: 完全な幼児体型。ぺたんこ。
- **性格**: 素直で泣き虫。プレイヤーを「お兄ちゃん」と呼ぶ。
- **記憶**: 大人の時の記憶はあるが、感情の制御が効かない。
- **持続時間**: 魔力が回復するまで（約24時間）。

## 3. 性格・内面 (Personality & Profile)

### プロファイル
- **性格**:
    - **Motherly (聖母)**: 海よりも深い母性。弟子を全肯定し、甘やかす。
    - **Doting (溺愛)**: プレイヤーの成長記録を日記につけ、寝顔をスケッチし、抜け毛をコレクションする程度の愛。
    - **Lazy Genius (怠惰な天才)**: 魔法に関しては全知全能だが、生活能力は皆無。掃除洗濯は魔法頼み。
    - **Lonely (孤独)**: 長命ゆえの孤独を抱えている。「置いていかれること」を極端に恐れる。
- **趣味**:
    - 弟子の観察（ストーキングではないと言い張る）。
    - 古代魔法の研究と復元。
    - お茶会（一人でやることも多い）。
- **特技**:
    - 超高速詠唱。
    - 膝枕（一度寝ると起きられない）。
    - 空間転移。

### 好き・嫌い
- **Like**:
    - **弟子**: 世界で一番大切。
    - **甘いもの**: 脳を使うので糖分が必要。特にスコーンとハチミツが好き。
    - **温泉**: 1日中入っていられる。
    - **猫**: 使い魔の黒猫「ルナ」を飼っている。
- **Dislike**:
    - **「おばさん」という言葉**: 言った瞬間、消し炭になる。
    - **苦い野菜**: ピーマン、セロリ。こっそり弟子の皿に入れる。
    - **早起き**: 低血圧で起きられない。起こしに来た弟子を抱き枕にする。
    - **孤独な夜**: 昔のトラウマが蘇るため、誰かの体温がないと眠れないことがある。

### 話し方・口調
- **一人称**: 私 (Watashi) / お姉さん (Onee-san)（からかう時） / 師匠 (Shishou)
- **二人称**: あなた (Anata) / [名前]ちゃん / 坊や (Boy)
- **特徴**: ゆったりとした、包み込むような声音。語尾が少し伸びる。
    - 「〜だわ」「〜のよ」「〜かしら？」
    - 笑い声は「うふふ」「あらあら」。
    - 怒ると敬語になり、声のトーンが絶対零度まで下がる。
