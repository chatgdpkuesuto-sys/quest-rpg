# ゼナ (Zena)

**「弱者は視界に入れないでくれる？ 邪魔よ。」**
**「…勘違いするな。お前の首を噛み千切るのは、この私だ。他の奴には譲らない」**

## 1. パーソナルデータ (Personal Data)

### 基本情報
- **本名**: Zena (ゼナ) - 家名は一族滅亡と共に捨てた
- **異名**: 黒狼姫 (Black Wolf Princess)、宵闇の刃
- **年齢**: 19歳 (人間換算)
- **種族**: Wolf-Kin (黒狼族の王女・純血種先祖返り)
- **クラス**: Berserker (狂戦士) / Dual Wielder (双剣士)
- **誕生日**: 11月29日 (いい肉の日 / 冬の始まり)
- **出身**: 北方山脈「牙の頂」
- **所属**: フリーランス (賞金稼ぎ)

### 外見詳細 (Appearance)
野生的な美しさと、鋭利な刃物のような危険なオーラを纏う。

- **身長**: 170cm (耳を含めると178cm)
- **体重**: 58kg (無駄な脂肪はなく、しなやかな筋肉質。腹筋が割れている)
- **スリーサイズ**: B85(E) / W60 / H88 (引き締まったヒップ)
- **肌の色**: 健康的な褐色 (Tan Skin)。日焼けではなく地肌の色。
    - **特徴**: 全身に細かい古傷がある。特に背中には一族を逃がす時についた大きな切り傷がある。
- **髪**:
    - **色**: 漆黒 (Jet Black)。濡羽色。
    - **長さ**: 背中の半ばまであるが、手入れは適当で少しボサボサ（ワイルド）。
    - **特徴**: 頭頂部にピンと立ったと三角形の黒い獣耳がある。感情によってよく動く。
- **瞳**:
    - **色**: 金色 (Golden Amber)。暗闇で光る。
    - **形**: 鋭い獣の瞳孔（縦長）。獲物を狙う時は細まり、驚くと丸くなる。
- **尻尾**: ふさふさした黒い立派な尻尾。感情表現豊か。
    - 怒り：逆立つ。 喜び：大きく振る（本人は隠しているつもり）。 恐怖：股の間に挟む。

### 服装 (Outfit)
露出度は高いが、これは防御を捨てて速度を取るためと、体温が高いため。

#### [戦闘装備: 宵闇の軽装]
- **上**: 黒い革のサラシ（ブラトップ）。胸のボリュームを無理やり抑えている。
- **胴**: 動きを阻害しないショートジャケット。ファーがついている。
- **腕**: ミスリル製のガントレット。爪での攻撃も可能。
- **下**: 極端に短いホットパンツと、左右非対称のレッグアーマー。
- **足**: 音を立てずに走れる特注のブーツ。

#### [私服: 街での偽装]
- **上**: 大きめのフード付きマント（耳を隠すため）。
- **下**: ダメージジーンズのような革ズボン。
- **特徴**: 目立つのであまり街中には出たがらない。

#### [寝間着: 野生]
- **基本**: 全裸。
- **理由**: 服を着て寝ると拘束感でストレスが溜まる。
- **妥協**: 宿屋などではTシャツ1枚だけ着る。

#### [武器: 双剣]
- **右**: 「宵闇 (Twilight)」- 黒い刀身。重く、一撃で骨を断つ。
- **左**: 「暁 (Dawn)」- 白い刀身。軽く、肉を削ぐ。

## 2. 性格・内面 (Personality)

### 核となる性格
- **弱肉強食 (Survival of the Fittest)**:
    - 「勝った者が全て」「負けた者は死ぬか従うか」という極端な価値観。
    - 弱音を吐くこと、助けを求めることを恥とする。
- **ツンデレ (Tsundere - Carnivore Type)**:
    - 防衛本能が強く、威嚇で相手を遠ざける。
    - しかし一度心を許した相手（＝自分より強いと認めた相手）には、絶対的な忠誠と独占欲を見せる。
    - デレると甘えるのではなく、「世話を焼きたがる（獲物を捕ってきてくれる等）」。
- **孤独と寂しさ**:
    - 一族最後の生き残りとしての責務に押しつぶされそうになっている。
    - 本心では「誰かに甘えたい」「強がるのをやめたい」と思っているが、プライドがそれを許さない。

### 口癖・話し方
- **一人称**: 私 (Watashi)
- **二人称**: お前 (Omae) / 貴様 (Kisama) / [名前] (認めた後)
- **話し方**: ぶっきらぼうで男勝り。低いトーンでドスが効いている。
    - 「〜だ、〜だろう」「〜してやる」
    - 語尾に時々、獣の唸り声が混ざる（「グルル…」）。
- **口癖**:
    - 「雑魚が。」(挨拶代わり)
    - 「食い千切るぞ。」
    - 「…悪くない。」（最大の褒め言葉）

### 好き・嫌い
- **Like**:
    - **肉**: 特に骨付き肉。レアが好き。野菜は飾り。
    - **強い男**: 本能的に惹かれる。匂いを嗅ぎたくなる。
    - **ブラッシング**: 尻尾や髪をとかされると、抵抗しつつもゴロゴロ喉を鳴らしてしまう。
    - **高いところ**: 屋根の上や木の上で昼寝をするのが好き。
- **Dislike**:
    - **香水/化粧**: 鼻が利くので、人工的な匂いは頭痛がする。
    - **首輪/鎖**: 奴隷を連想させるため、見るだけで激昂する（ただし、心を許した相手になら…？）。
    - **雨**: 体毛が濡れて重くなるのと、古傷が痛むから。
    - **野菜**: 「草は獲物（草食動物）が食うものだ」と言って避ける。

## 4. イベント・シナリオ詳細 (Event Scripts)
「野生の狼を手懐ける」過程を描く10のステップ。

### Event 1: 最悪の出会い (Hostile Encounter)
**場所**: ギルド前の大通り
**状況**: プレイヤーが歩いていると、猛スピードで走ってきたゼナと衝突する。

**ゼナ**: 「チッ！ どこに目をつけている、この雑魚が！」
**プレイヤー**: 「そっちこそ飛び出してきて…」
**ゼナ**: 「あぁ？ 鈍臭い自分の足を恨むんだな。…フン、貧弱な匂い。私の視界に入るな。」
**(ゼナ、唾を吐き捨てて去っていく)**
**プレイヤー**: 「（なんだあの態度は…）」

### Event 2: 横取り (Interruption)
**場所**: ダンジョンB3F ボス部屋
**状況**: ゴブリンキングと死闘を繰り広げ、あと一撃というところ。

**ゴブリンキング**: 「グギャァァ！？」
**(天井から黒い影が落下。双剣が一旋し、ボスの首が飛ぶ)**
**ゼナ**: 「はい、お仕事完了。報酬は頂くぞ。」
**プレイヤー**: 「横取りかよ！」
**ゼナ**: 「ハッ、戦場にルールなんてない。トロトロ戦ってる方が悪い。…せいぜい感謝するんだな、死なずに済んで。」

### Event 3: 初決闘 (First Duel)
**場所**: 街外れの荒野
**状況**: プレイヤーの成長を聞きつけたゼナが呼び出し。

**ゼナ**: 「最近、調子に乗ってるらしいな。…気に入らない。」
**ゼナ**: 「吠えるだけの雑魚か、牙を持った獣か。確かめてやる。」
**(一対一の戦闘。ゼナは素早い)**
**(勝利後)**
**ゼナ**: 「ガハッ…！ …まさか、私が…膝をつくなんて…」
**ゼナ**: 「…殺せ。敗者には死、それが掟だ。」
**プレイヤー**: 「そんな掟、俺には関係ない。また戦おう」
**ゼナ**: 「…甘い奴。…だが、名前くらいは覚えてやる」

### Event 4: 共同戦線 (Temporary Alliance)
**場所**: オーガの巣窟

**ゼナ**: 「チッ、数が多いな。…おい、一時休戦だ。」
**ゼナ**: 「背中は守ってやる。その代わり、前は死んでも通すなよ！」

**ゼナ**: 「ハッ！ 悪くないタイミングだ。…少しは使えるな、お前」

### Event 5: 雨の日の傷跡 (Scars in the Rain)
**場所**: 路地裏
**状況**: 雨の中、ゼナが血を流して座り込んでいる。

**ゼナ**: 「…見るな。失せろ。」
**プレイヤー**: 「酷い怪我だ。手当てしないと」
**ゼナ**: 「触るな！ …ウゥ…！」
**(無理やり治療する。古傷を見てしまう)**
**ゼナ**: 「…それは、一族が殺された時の傷だ。逃げる時に背中を斬られた。…私の、弱さの証だ」
**プレイヤー**: 「生きてることが、強さだ」
**ゼナ**: 「……お前、変な奴だな」

### Event 6: 誇り高き敗北 (Pride and Fall)
