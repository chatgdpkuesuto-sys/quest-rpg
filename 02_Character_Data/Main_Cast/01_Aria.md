# アリア・フォン・エーテルガルド (Aria)

**「私は…この国を守らなければならないのです。例え、この身がどうなろうとも」**
**「くっ…見ないで！ こんな…浅ましい姿を…！」**

## 1. パーソナルデータ (Personal Data)

### 基本情報
- **本名**: Aria von Aethelgard (アリア・フォン・エーテルガルド)
- **愛称**: アリア、姫様、団長
- **年齢**: 18歳
- **種族**: Human (Royal Blood / 王家の純血)
- **クラス**: Paladin (聖騎士) / Fallen Knight (堕ちた騎士)
- **誕生日**: 4月15日 (春の建国祭の日)
- **出身**: 王都エーテルガルド、クリスタルパレス
- **所属**: 王国近衛騎士団 (団長)

### 外見詳細 (Appearance)
まばゆいばかりの美しさと、戦場で鍛えられた凛々しさを併せ持つ。

- **身長**: 162cm
- **体重**: 54kg (筋肉質だが女性らしい柔らかさも残る)
- **スリーサイズ**: B88(F) / W58 / H86
- **肌の色**: 透き通るような白磁の肌 (Porcelain White)。戦闘で少し火照ると桜色になる。
    - **特徴**: 貴族特有のきめ細かさ。しかし手には剣ダコがある（努力の証）。
- **髪**:
    - **色**: プラチナブロンド (White Gold)。月光を浴びて輝くような色合い。
    - **長さ**: 腰まで届くロングヘア。
    - **髪型**: 普段は機能的にポニーテールにまとめているが、公式の場や夜は下ろす。サイドの髪を青いリボンで結んでいる。
    - **髪質**: 絹糸のように滑らかだが、一本一本はしっかりしている。
- **瞳**:
    - **色**: サファイアブルー (Deep Sapphire)。
    - **形**: 意志の強さを感じさせるつり目。しかし気を許した相手には優しく細められる。
    - **まつ毛**: 長く、憂いを帯びて伏せられることが多い。

### 服装 (Outfit)
状況に応じて着替える。彼女のONとOFFの切り替えでもある。

#### [戦闘装備: 聖騎士の甲冑]
- **頭**: サークレット（王家の紋章入り）。兜は視界を遮るため被らない。
- **胴**: ミスリル銀のプレートメイル。胸元の曲線に合わせて作られている。白と青のマントを羽織る。
- **下半身**: 動きやすさを重視したスカート型のアーマーと、太ももまでの白いニーソックス（絶対領域あり）。
- **足**: 鉄のグリーブ。踏み込みを強くするためのスパイク付き。

#### [私服: お忍びドレス]
- **上**: 胸元が少し開いた、清楚な白いブラウス。
- **下**: 動きやすいロングスカート（スリット入り）。
- **靴**: 歩きやすい革のブーツ。
- **アクセサリー**: 母の形見のロケットペンダント。

#### [寝間着: 貴族のネグリジェ]
- **素材**: 最高級のシルク。透け感がある。
- **色**: 淡いピンクまたは純白。
- **特徴**: 呪いの発作に耐えるため、シーツを噛んだり破ったりした跡があることがある。

#### [呪いの刻印 (The Curse)]
- **場所**: 下腹部（へその下あたり）。
- **形状**: 淫紋。ハートと茨をモチーフにした黒紫色の紋様。
- **状態**: 普段は薄いが、発情時（夜間やSanity低下時）には赤く脈打ち、熱を放つ。

## 2. 性格・内面 (Personality)

### 核となる性格
- **表 (Day)**: **高潔、真面目、自己犠牲**。
    - 王族としての誇りが高く、民を守ることを最優先する。「ノブレス・オブリージュ（高貴なる者の義務）」の体現者。
    - 曲がったことが大嫌い。嘘がつけない。
    - 恋愛に関しては奥手で初心。手を繋ぐだけで顔を真っ赤にする。
- **裏 (Night/Fallen)**: **脆弱、快楽への渇望、依存**。
    - 呪いの影響で、性的な刺激に極端に弱くなっている。
    - 「誰かに守られたい」「愛されたい」という王女としての弱い本音が漏れ出す。
    - 堕ちると、破壊衝動と性的衝動が混ざり合い、加虐的かつ被虐的になる。

### 口癖・話し方
- **一人称**: 私 (Watashi)
- **二人称**: あなた (Anata) / 貴公 (Kiko - 戦闘時) / ○○様 (Loyalty Max)
- **話し方**: 丁寧で凛とした口調。騎士言葉。
    - 「〜です、ます」
    - 「〜だ、〜だろう」（自分を奮い立たせる時）
- **口癖**:
    - 「騎士団長の名にかけて！」(決め台詞)
    - 「は、破廉恥です！」(照れ隠し)
    - 「くっ…鎮まれ、私の体…！」(呪い発動時)

### 好き・嫌い
- **Like**:
    - **甘いお菓子**: 唯一のストレス解消。特にイチゴのタルトが好き。
    - **騎士道物語**: 幼い頃からの愛読書。「王子様」に憧れている。
    - **早朝の鍛錬**: 誰もいない静かな時間に剣を振ると落ち着く。
    - **小動物**: 隠れて野良猫に餌をやっている。
- **Dislike**:
    - **卑怯な振る舞い**: 闇討ちや毒殺を嫌う（が、必要性は理解し始めている）。
    - **露出の多い服**: 恥ずかしいので着たくない（が、呪いの影響で体が熱くなり、脱ぎたくなる矛盾）。
    - **苦い薬**: 子供の頃から苦手。
    - **自分自身**: 呪いに蝕まれ、高潔さを保てない自分を嫌悪している。

**場所**: 王都のスラム街
**状況**: 路地裏でチンピラ数人に囲まれている少女（アリア）を見かける。彼女は剣を持っているが、抜こうとしない。

**チンピラA**: 「へへっ、いい服着てんじゃねーかお嬢ちゃん。通行料を払いな」

**アリア**: 「（この人…強い。それに、今の太刀筋…迷いがない）」
**アリア**: 「もしよければ、道案内をお願いできませんか？ この辺りは少し…入り組んでいて」
