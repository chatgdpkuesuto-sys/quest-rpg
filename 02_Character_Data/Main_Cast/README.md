## 📊 ステータスシステム (6-Stat System)
キャラクターは以下の6つの能力値で定義されます。

| Stat | 意味 | 主な判定 |
|:---:|:---|:---|
| **Power** | 筋力、破壊力 | 近接攻撃、重い物を持ち上げる |
| **Speed** | 敏捷性、反応 | 回避、先制、手先の早業 |
| **Tough** | 耐久力、根性 | HP、毒耐性、苦痛への抵抗 |
| **Mind** | 知力、魔力 | 魔法攻撃、知識、精神抵抗 |
| **Charm** | 魅力、話術 | 交渉、誘惑、情報の引き出し |
| **Skill** | 技術、器用さ | 精密作業、製作、解除 |
