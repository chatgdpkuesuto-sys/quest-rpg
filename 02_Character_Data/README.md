# 02_Character_Data

キャラクターの確定データ（ステータス、設定、イベント）を管理するディレクトリです。

## 📂 ディレクトリ構成

### **[`Main_Cast/`](Main_Cast/)**
プレイヤーのパートナーとなる主要ヒロインのデータ。
*   **統合ファイル**: **[`01_Main_Cast.md`](Main_Cast/01_Main_Cast.md)** に全5人のヒロインデータが集約されています。

### **[`Sub_Cast/`](Sub_Cast/)**
ストーリーに関わる準主要キャラクター（サブヒロイン、ライバル、師匠キャラなど）。

### **[`NPCs/`](NPCs/)**
街の人々、使い捨ての依頼人、店主などの汎用NPCデータ。

---

## 📝 キャラクター作成ルール
新規キャラクターやNPCを作成する際は、Core Engine の以下のルールに従ってください。
*   **作成ルール**: [`../00_Core_Engine/06_Character_Creation.md`](../00_Core_Engine/06_Character_Creation.md)
*   **ステータス**: Power, Speed, Tough, Mind, Charm, Skill (6-Stat System)
