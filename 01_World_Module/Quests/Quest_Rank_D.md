# Rank D クエスト (Intermediate)

**「新米卒業おめでとう。ここからが本当の冒険だ。」**

## D-001: 廃鉱山のゴースト討伐 (Ghost Hunt)
- **依頼主**: 鉱山街の親方
- **報酬**: 500G + 銀鉱石 x3
- **場所**: 旧鉱山地区 (Old Mine)
- **内容**: 鉱山に住み着いた悪霊（ゴースト）を退治してほしい。物理攻撃が効かないので、魔法か聖水が必要。
- **ヒント**: 聖水は街の教会で買える。

## D-002: 商隊護衛 (Caravan Escort)
- **依頼主**: 行商人の組合
- **報酬**: 800G + 珍しいスパイス
- **場所**: 街道 (Main Road) -> 隣町
- **内容**: 盗賊団が出没する街道を護衛する。途中で大規模な襲撃イベントが発生する。
- **敵**: バンデットリーダー (Lv5) とその手下。

## D-003: 幻のキノコを探せ (Mushroom Hunt)
- **依頼主**: 偏屈な錬金術師
- **報酬**: ハイポーション x5 + 爆弾のレシピ
- **場所**: 迷いの森 (Deep Forest)
- **内容**: 「マンドラゴラ」を採取する。引き抜くと悲鳴を上げて気絶させてくる（CON判定 DC12）。
- **注意**: 夜にしか生えていない。

## D-004: 下水道の主 (Sewer Lord)
- **依頼主**: 王都の衛生局
- **報酬**: 1000G + 汚れた鍵
- **場所**: 王都地下水道 (Capital Sewers)
- **内容**: 下水道に巨大ワニ（ジャイアントクロコダイル）が住み着いている。実は誰かが捨てたペットらしい。
- **ボス**: ジャイアントクロコダイル (Lv8)

## D-005: 古代遺跡の調査 (Ruins Survey)
- **依頼主**: 考古学者ギルド
- **報酬**: 600G + 未鑑定のアーティファクト
- **場所**: 砂漠の遺跡 (Sand Ruins)
- **内容**: まだ誰も入っていない遺跡の入り口が見つかった。内部の地図を作成し、持ち帰ること。
- **ギミック**: 床が抜ける罠、転がる大岩。

## D-006: 呪われた人形 (Cursed Doll)
- **依頼主**: 裕福な商人
- **報酬**: 2000G (呪い解除費含む)
- **場所**: 商人の屋敷
- **内容**: 娘が拾ってきた人形が夜な夜な動き出し、家の人を襲う。人形の正体は悪魔憑き。
- **ボス**: デーモンドール (Lv6)

## D-007: グリフォンの卵 (Griffin Egg)
- **依頼主**: 王立騎士団
- **報酬**: 1500G + 騎士団の推薦状
- **場所**: 高山 (High Mountains)
- **内容**: グリフォンの巣から卵を一つ盗んでくる。親に見つかると即死級の攻撃が飛んでくる。隠密 (Stealth) 推奨。
- **選択**: 卵を盗まずに親を倒すと、報酬がなくなる（騎士団は騎獣として育てたい）。

## D-008: 偽金作りを追え (Counterfeit)
- **依頼主**: 衛兵隊長
- **報酬**: 1000G + 押収品の一部
- **場所**: 港の倉庫街
- **内容**: 偽金を作っている組織のアジトを突き止める。戦闘よりも潜入・捜査がメイン。
- **判定**: `Investigation (INT)` や `Thieves' Tools (DEX)` が活躍。

## D-009: 吟遊詩人の護衛 (Bard Tour)
- **依頼主**: 人気吟遊詩人
- **報酬**: 300G + 詩人の歌によるバフ (3日間、幸運+1)
- **場所**: 巡業ルート
- **内容**: ストーカーファンやモンスターから詩人を守りながら、3つの村を回る。
- **イベント**: 詩人の歌で隠し通路が開くことがある。

## D-010: 魔剣の試練 (Sword Trial)
- **依頼主**: 謎の老人
- **報酬**: 魔剣「ブラッドエッジ」 (呪い付き)
- **場所**: 試練のの洞窟
- **内容**: 老人が持つ剣を引き抜くことができるか試される。剣に認められるには、自分の血を捧げ、精神世界で剣の精霊と戦わなければならない。
- **ボス**: 剣の精霊 (Lv10) - 物理無効、精神攻撃主体。
