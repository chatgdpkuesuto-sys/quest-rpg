# 🌅 Prologue: The Fated Choice (運命の選択)

**GM**: あなたは気がつくと、見知らぬ土地に立っていた。
記憶はおぼろげで、ここがどこなのかも分からない。
ただ、どこか懐かしいような、それでいて異質な風が頬を撫でる。

その時、**5人の少女たち**があなたの前に現れる。
彼女たちはまだお互いを知らないが、運命の糸であなたと繋がっている。

---

## 🎲 Selection Phase (運命のパートナー選択)

**GM**: さあ、この世界での最初の「相棒（パートナー）」を**一人だけ**選びたまえ。
選ばなかったヒロインとも、後の冒険で出会うことができる。

### Option A: 🛡️ The Fallen Knight (アリア)
**「困っているのですか？ …私の手を取ってください」**
- **Role**: **Tank / Paladin** (守備・物理)
- **特徴**: 高潔だが押しに弱い騎士。真面目すぎる性格をイジりたい人向け。
- **Start**: 王道ヒロインルート。

### Option B: 🐺 The Black Wolf (ゼナ)
**「フン、弱そうな奴。…私が守ってやるよ」**
- **Role**: **DPS / Berserker** (攻撃・敏捷)
- **特徴**: ツンデレで野性的な獣人。主従関係（飼い犬・飼い主）を楽しみたい人向け。
- **Start**: 凸凹コンビルート。

### Option C: 🧙‍♀️ The Azure Witch (エララ)
**「あらあら、迷子ね。…お姉さんが助けてあげる」**
- **Role**: **Magic / Sage** (魔法・範囲)
- **特徴**: 包容力お化けの師匠。甘やかされたい、バブみを感じたい人向け。
- **Start**: 年上同棲ルート。

### Option D: 🧪 The Mad Scientist (エリーゼ)
**「君、いい身体してるね…実験、付き合わない？」**
- **Role**: **Healer / Alchemist** (回復・支援)
- **特徴**: 狂気のマッドサイエンティスト。M気質、変態プレイを楽しみたい人向け。
- **Start**: 監禁・実験ルート。

### Option E: 🐱 The Catgirl Thief (ユニ)
**「にゃはは！ お金ないなら、身体で払って？」**
- **Role**: **Utility / Rogue** (探索・金策)
- **特徴**: 守銭奴な猫耳盗賊。貢がせたい、借金生活を楽しみたい人向け。
- **Start**: 借金返済ルート。

## 🎭 Encounter Phase (運命の出会い)

選んだパートナーとの「最初の出会い」が再生される。

### If Selected **[Aria]** - The Fallen Knight
**Situation**: 街の広場で貧血を起こした彼女を支える。
**Aria**: 「顔色が悪いですね。…立てますか？ 逆に私が肩を貸しましょう」
- **Choice 1 (Accept)**: 「ありがとう、優しいね」と素直に甘える。 -> **Route A (Mutual)**
- **Choice 2 (Hesitate)**: 「大丈夫だ」と強がり、距離を取ろうとする。 -> **Route B (Abduction)**

### If Selected **[Zena]** - The Black Wolf
**Situation**: 路地裏で絡まれているところを助けられる（あるいはカツアゲされる）。
**Zena**: 「フン、ふらふらして…みっともない。…おい、いい匂いがするな、お前」
- **Choice 1 (Confront)**: 「お前こそ、いい匂いだ」と睨み返す。 -> **Route A (Mutual)**
- **Choice 2 (Retreat)**: 「関わらないでくれ」と背を向けて逃げる。 -> **Route B (Abduction)**

### If Selected **[Elara]** - The Azure Witch
**Situation**: 森で迷ったあなたを保護する。
**Elara**: 「ふふ、可愛い子。お姉さんが『いいところ』へ連れて行ってあげましょうか？」
- **Choice 1 (Trust)**: 「お願いするよ」と手を握る。 -> **Route A (Mutual)**
- **Choice 2 (Doubt)**: 「怪しいな」と警戒して後ずさる。 -> **Route B (Abduction)**

### If Selected **[Elize]** - The Mad Scientist
**Situation**: 行き倒れていたあなたを「素材」として拾う。
**Elize**: 「あら、まだ動けるの？ 手間が省けたわ。研究室はそっちよ」
- **Choice 1 (Interest)**: 「研究…？ 面白そうだ」と興味を示す。 -> **Route A (Mutual)**
- **Choice 2 (Run)**: 「ヤバい匂いがする」と逃げ出す。 -> **Route B (Abduction)**

### If Selected **[Yuni]** - The Catgirl Thief
**Situation**: 財布をスろうとして、逆にあなたの匂いに惹かれる。
**Yuni**: 「あっ、起きた！ ごめんごめん、盗むつもりじゃ…へへ、お詫びにイイことしてあげる？」
- **Choice 1 (Accept)**: 「…イイこと？」と期待して頷く。 -> **Route A (Mutual)**
- **Choice 2 (Refuse)**: 「信用できないな」と手を払う。 -> **Route B (Abduction)**

## 📜 Encounter Scenarios: The First Night (運命の初夜)

### Route A: Mutual Consent (合意・協力)
プレイヤーの適切な対応により、ヒロインは理性を保ちつつ（あるいは理性が飛んでも好意的に）、あなたを受け入れる。
-> **既存の「救済・支配・魔力供給」イベントへ進行（成功扱い）。**

---

### Route B: Abduction & Reversal (拉致と逆転)
**GM**: あなたの拒絶は、逆に彼女たちの「捕食本能」に火をつけてしまった！

#### 1. The Abduction (お持ち帰り)
- **Aria**: 「逃がしません…！ あなたは、保護対象（わたしのもの）です！」
    - 無理やり騎士団の独房へ連行され、手錠をかけられる。
- **Zena**: 「逃げる獲物は、追いかけたくなるんだよぉ！」
    - 全力疾走で追いつかれ、押し倒されてマウントを取られる。
- **Elara**: 「あらあら、悪い子にはお仕置きが必要ね…『テレポート』！」
    - 一瞬で地下実験室へ転送され、愛の鎖に繋がれる。

#### 2. The Crisis (襲撃)
ヒロインは欲望を剥き出しにして、あなたを貪ろうとする。
**Heroine**: 「もう我慢できない…全部、私のものにしてあげる…♥」

#### 3. 🧠 Tactical Choice (起死回生の一手: 弱点を攻めろ)
彼女の暴走を止めるには、**「弱点 (Weak Point)」** を的確に突く必要がある。
正解の選択肢（弱点）を選べば、**確定で「Critical (完全逆転)」** となる。

**【Vs Aria: 弱点 = 脇腹 / 鎧剥ぎ / 奉仕命令】**
- **A**: 「やめろ！ 騎士団長ともあろう者が！」（説教 -> 失敗）
- **B**: **「鎧の隙間から指を入れ、脇腹をくすぐる」**（**弱点: 脇腹** -> **Critical**）
- **C**: 「好きにしていいよ」（諦め -> 幸福な敗北）

**【Vs Zena: 弱点 = 肉球 / お手・お座り / マウント】**
- **A**: 「痛いのは嫌だ、優しくしてくれ」（哀願 -> 失敗）
- **B**: **「ドスの効いた声で『お座り！』と命じる」**（**弱点: 命令** -> **Critical**）
- **C**: 死んだふりをする。（回避 -> 幸福な敗北）

**【Vs Elara: 弱点 = 耳の先端 / 乳輪 / 放置プレイ】**
- **A**: **「耳元に息を吹きかけ、甘噛みする」**（**弱点: 耳の先端** -> **Critical**）
- **B**: 「助けて！ 吸い尽くされる！」（悲鳴 -> 失敗）
- **C**: 全力で暴れて抵抗する。（物理 -> 幸福な敗北）

**【Vs Elize: 弱点 = 足の裏 / 検体台 / 排泄許可】**
- **A**: 「この身体、君の研究に役立つはずだ」（協力 -> 失敗）
- **B**: **「逆に彼女を検体台に押し倒し、拘束する」**（**弱点: 検体台** -> **Critical**）
- **C**: 「やめてくれ！」（懇願 -> 幸福な敗北）

**【Vs Yuni: 弱点 = 首筋 / 尻尾責め / 拘束姦】**
- **A**: **「首筋を強く掴み、動きを封じる」**（**弱点: 首筋** -> **Critical**）
- **B**: 「お金ならいくらでもあげるよ」（金で解決 -> 失敗）
- **C**: 「警察に通報するぞ！」（脅迫 -> 幸福な敗北）

#### 4. The Resolution (判定結果: どっちに転んでも美味しい)

どのような結果になろうと、あなたはこの夜、ヒロインと深い絆（と快楽）を得る。

- **【Critical (正解: 攻めの一手)】**: 
    - あなたは主導権を握り、彼女を悦ばせて分からせる。
    - **Result**: **Dominant Lover (支配的な愛)**。彼女はあなたに夢中になり、尽くす喜びに目覚める。
    - **Bonus**: **Exp +100** (快楽の主導権ボーナス)

- **【Selection Failure (不正解: 運任せ)】**: 
    - **Luck Check** (`2d6 + LUCK` vs 10) を行う。
    - **成功**: なんとか和解し、対等な関係に持ち込む (Route Aへ)。
    - **失敗/ファンブル**: **Sweet Surrender (幸福な敗北)**。

- **【Sweet Surrender (幸福な敗北: 受けの一手)】**: 
    - **内容**: 抵抗虚しく、彼女の愛欲に溺れる。
    - **Benefit**: **MaxSP +1** (魔力回路の結合による恒久強化)。
    - **Status Change**: **Beloved (溺愛)**。

### 【Scene C: エララとの「魔力充填」】 (Elara: The Mana Transfer)
**場所**: 教会の懺悔室
**状況**: エララが顔面蒼白で倒れかかる。彼女の生命維持に必要な魔力が枯渇していた。

1.  **The Thirst**:
    - **Elara**: 「…だめ、意識が…消えちゃう…。坊や、お願い…私に『命』をちょうだい…」
    - **Elara**: 「濃厚なのがいいの…あなたの精気を、たっぷり絞り出して…♥」
2.  **Action (R-18)**:
    - **魔力注乳 (Breast Feed)**: 豊満な胸に顔を埋め、魔力を吸わせる（あるいは注ぐ）。
    - **子宮充填 (Womb Fill)**: 最奥にて全てを注ぎ込む。
3.  **Result**: 
    - 全盛期の魔力を取り戻したエララは、あなたを「最高の魔力供給源（スポンサー）」として溺愛し、離さなくなる。
    - **Get Ally**: **[Elara (Mentor)]** (Status: **Addicted**)

### 【Scene D: エリーゼとの「身体検査」】 (Elize: The Examination)
**場所**: 地下の研究室
**状況**: あなたは実験台の上にいる。エリーゼが妖しい薬品を持って近づいてくる。

1.  **The Experiment**:
    - **Elize**: 「素晴らしいわ。魔力回路がこんなに太いなんて…解剖するのが楽しみ」
    - **Elize**: 「ねぇ、中の具合はどうなってるの？ あなたの『遺伝子』、採取させてもらうわよ？」
2.  **Action (R-18)**:
    - **精液採取 (Sample)**: 彼女の手と器具による、執拗なまでの強制射精。
    - **交配実験 (Breed)**: 自らの身体を使って、受胎率を検証する。
3.  **Result**: 
    - 未知の快感を知ったエリーゼは、あなたを「生涯かけて研究すべきテーマ」と定義し、独占契約を結ぶ。
    - **Get Ally**: **[Elize (S-Tier)]** (Status: **Obsessed**)

### 【Scene E: ユニとの「借金返済」】 (Yuni: The Debt Payment)
**場所**: 安宿のベッド
**状況**: ユニがあなたの財布を返し、上目遣いで見つめてくる。

1.  **The Deal**:
    - **Yuni**: 「…お金、盗もうとしてごめん。でも、妹の薬代がなくて…」
    - **Yuni**: 「体で返すから…だから、アタシを警察には突き出さないで…！」
2.  **Action (R-18)**:
    - **優しくする (Gentle)**: 怯える彼女を優しく抱きしめ、愛を教える。
    - **躾ける (Train)**: 悪い手癖を治すため、徹底的にお仕置きする。
3.  **Result**: 
    - 借金の肩代わり（という名の保護）をしてくれたあなたに、ユニは絶対の忠誠と愛情を誓う。
    - **Get Ally**: **[Yuni (Rogue)]** (Status: **Pet**)

## 📌 Re-Encounter Rule (再会ルール)
選ばなかった（最初に遭遇しなかった）ヒロインとも、後の「街のイベント」で出会うことができる。
- **Aria**: ギルドで依頼を受けている時に遭遇。
- **Zena**: 夜の酒場で遭遇。
- **Elara**: 教会で治療を受ける時に遭遇。
