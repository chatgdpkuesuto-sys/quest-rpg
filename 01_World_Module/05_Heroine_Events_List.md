# 💘 Heroine Event Progression (ヒロイン攻略チャート)

各ヒロインとの関係性を深めるためのイベントリスト。
条件を満たした状態で「会話」「デート」「特定の場所へ行く」ことで発生する。

## 🌍 World Impact: Male Rarity (男性希少性の影響)
この世界では、男性は**「国家予算並みの価値がある資源」**である。
ヒロインイベントを進めることは、単なる恋愛ではなく**「誰がこの資源（あなた）を独占するか」**という政治的・生物的闘争を意味する。

### 1. 独占欲と嫉妬 (Jealousy)
- 特定のヒロインのLoveが高まると、他のヒロインからの**「妨害イベント」**が発生する。
- 例: アリアとデート中に、ゼナが乱入して決闘を申し込む。

### 2. 公衆の視線 (Public Gaze)
- 街を歩くだけで、モブ女性たちからの**「逆ナンパ（襲撃）」**判定(2d6)が発生する。
- 失敗すると路地裏に連れ込まれ、**R-18イベント（搾取）**が発生し、SPが枯渇する。
- これを防ぐには、強力なヒロイン（護衛）を連れて歩く必要がある。

### 3. 種付けの義務 (Breeding Duty)
- Loveが8を超えたヒロインとは、**「子作り」**が公的な義務となる。
- エンディングでは、何人の子供を残したかがスコアとなる。

### 4. 異界の加護 (Divine Protection of the Male)
貴方は世界で唯一の「特殊な魂」を持っているため、**「貴方に愛された女性（ヒロイン）」は運命的な加護を得る。**
- **不死 (Immortality)**: ヒロインは戦闘でHPが0になっても**絶対に死亡しない**。
- **生還 (Survival)**: たとえ敗北し、捕食され、行方不明になっても、必ず貴方のもとへ**生還**する。
    - ※ただし、一時的に「捕獲」されたり、酷い目に遭う（R-18イベント）可能性はある。心は折れない。

## 📌 Love System & Event Trigger (攻略システム)

### 1. Love Stat (親密度)
各ヒロインには **0〜10** の親密度(Love)が存在する。
- **初期状態 (Lv 1-5)**: **【隠した初恋 (Hidden First Love)】**
    - ヒロインは最初からプレイヤーに一目惚れしているが、立場や理性でそれを隠している。
- **転換点 (Lv 6)**: **【事実婚 / ダークらぶらぶ (Dark Love-Love)】**
    - 感情が爆発し、結婚してもおかしくない重い愛（ダーク寄り）へと変貌。
    - 感度が急上昇。視線や軽い接触だけで「即イき」するようになる。
- **到達点 (Lv 9)**: **【絶対愛 / 熟年超えの熱情 (Command Orgasm)】**
    - 熟年夫婦よりもアツアツな関係。愛が重すぎて逃げ場がない。
    - プレイヤーの「イけ」という命令だけで絶頂する「言魂絶頂 (Command Orgasm)」が可能。
- 詳細な情事描写: **[Main_Cast_Love_Scenes.md](file:///c:/Users/kuesu/GEM_Project_Root/04_Sexual_Events/Main_Cast_Love_Scenes.md)**

### 2. How to Raise Love (親密度の上げ方)
街での行動(`18_Town_Events.md`)で上昇させる。
- **デート (Date)**: Love +1 (1日1回まで)
- **プレゼント (Gift)**: Love +1 (高価なものなら+2)
- **クエスト同行 (Quest)**: ボス撃破時に生存で Love +1
- **Hイベント (Sexual)**: 濃厚な一夜を過ごすと Love +2 (ただしSP消費)

### 3. Event Trigger (イベント発生)
Loveが特定の閾値(1, 3, 5, 7, 9)に達した際、**次の行動開始時に強制発生**する。
GMは「LoveがXになったので、イベント『〜』が発生します」と宣言すること。
