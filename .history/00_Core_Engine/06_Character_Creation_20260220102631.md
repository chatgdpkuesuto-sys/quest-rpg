# キャラクター作成

**6能力値（筋力、敏捷、耐久、知性、魅力、技巧）** を使用した作成ルール。

## 1. 能力値の決定 (Stats)
以下のいずれかの方法で決定する。

### ポイント割り振り
ダッシュボード（Live Server）の初期画面で行う。

- **初期値**: 全て `0`
- **ポイント**: `3点` を自由に割り振る。
- **上限**: 初期作成時の上限は **`+3`** まで可能。
    - 例: 筋力+3, その他 0 (一点特化型)
    - 例: 筋力+1, 敏捷+1, 耐久+1 (バランス型)

### B. ダイスロール (Random Roll)
- `1d6` を3回振る。出た目に対応する能力値を `+1` する。
    - 1: Power
    - 2: Speed
    - 1: 筋力
    - 2: 敏捷
    - 3: 耐久
    - 4: 知性
    - 5: 魅力
    - 6: 技巧
    - ※重複した場合、その能力値は累積する（最大+3まで）。

## 2. 算出値

能力値決定後、以下を算出する。

- **最大HP**: `5 + 耐久`
- **最大SP**: `3 + 知性`

## 3. ジョブ選択
**能力値が「1以上」のステータスに対応するジョブのみ選択可能。**
参照: `[JobSystem]` (`13_Job_System.md`)

- **戦士**: 要 筋力 1+
- **盗賊**: 要 敏捷 1+
- **魔術師**: 要 知性 1+
- **吟遊詩人**: 要 魅力 1+
- **重装兵**: 要 耐久 1+
- **技師**: 要 技巧 1+

**Rule**: 条件を満たしていないジョブには就けない。ポイント割り振り時は計画的に行うこと。

## 4. 装備選択
ジョブに応じた初期装備セットを受け取る。

- **戦士セット**: 剣(dmg2), 革鎧(AC1)
- **盗賊セット**: 短剣(dmg1), 投げナイフ, 革鎧(AC1)
- **魔術師セット**: 杖(dmg1), ローブ(AC0), ポーション
- **共通**: 冒険者セット (松明、保存食、ロープ)

## 5. パーソナリティ (Personality)
- **名前**: 自由に決定。
- **動機**: なぜ冒険に出たか (金、名声、復讐、好奇心...)。
