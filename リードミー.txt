# GEM Fantasy RPG Project: Knowledge Repository

## 📌 Project Overview
このリポジトリは、**「Folder-as-Knowledge（フォルダ構造そのものを知識とする）」**アーキテクチャを採用したTRPGシステムです。
AI（GM）は、このディレクトリ構造を「脳の引き出し」として使用し、必要なルール・データ・キャラクター設定を動的に参照してセッションを進行します。

---

## 🚀 How to Start
1. **Initialize**: `00_Core_Engine/00_Principles.md` を読み込み、ゲームモードを決定。
2. **Create Char**: `13_Job_System.md` からクラスを選択し、キャラを作成。
3. **World Entry**: `World_Setting.md` の「ノワール」から冒険を開始。
4. **Action**: `Command_Manual.md` に従い、行動を宣言。

> **Note for Developers**:
> 新しいファイルを追加する際は、必ず `00_Core_Engine/00_Principles.md` のフォーマットに従い、Frontmatter (YAMLヘッダ) を付与してください。
