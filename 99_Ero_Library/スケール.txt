## 0. 世界法則: 魔力質量保存則 (World Law: Mana-Mass Correlation)
この世界では、**「魔力保有量 (Mana Capacity)」が「身体サイズ (Body Size)」に直結する**。
強大な魔力を持つ者は必然的に巨体となり、魔力を失えば縮小する。

*   **Law**: High Mana = Giant Body / Low Mana = Small Body.
*   **Dynamic**: 戦闘や絶頂（魔力放出）により、リアルタイムでサイズが変動する。
*   **Exception**: 一部の種族（エルフ、ドワーフ等）や、「圧縮魔法」の使い手は例外として人間サイズを保つ。

## 2. 文脈的選択ロジック (Contextual Selection Logic)
# ランダムではなく、敵種族や「その場のムード」に合わせてクラスを自動選択せよ。

### Rule A: 種族・役割によるデフォルト (By Species)
*   **God / Ancient Dragon / Castle Golem**: -> **[Class 6: Colossus]**
*   **Dragon / Demon Lord / Troll**: -> **[Class 5: Titan]** or [Class 4]
*   **Valkyrie / Amazon / Knight**: -> **[Class 4: Tall]**

### Rule C: 特殊な変異 (Mutations)
*   **[Gigantification]**: 魔法暴走やボスの「本気モード」 -> **強制的に Class 5 (Titan) 化**。
*   **[Minimalist]**: 呪い、罠、敗北ペナルティ -> **強制的に Class 1 (Micro) 化**。
*   **[Mana Drain Shrink]**: 主人公の精液は魔力を中和する。絶頂と共に巨体が収縮し、**無力な1mサイズ（Class 2）**へと成り下がる。同時に**精神も幼児退行（IQ低下）**し、完全に「メスガキ」化する。

### Rule D: 捕獲と支配 (Capture & Control)
*   **[Submission]**: 魔力を失い縮小した対象は、無力感と共に**主人公を「ご主人様 (Master)」と認識**し、絶対服従するペットとなる。
*   **[Womb Crest (淫紋)]**: 屈服の証として、下腹部（子宮位置）に**ハート型の淫紋**が浮き出る。
    *   *Visual*: ピンク色に発光する幾何学模様。興奮すると明滅し、所有者の魔力を受け入れるポートとなる。
*   **[Size Control]**: 淫紋を刻まれた対象のサイズは、**ご主人様の意思（魔力注入）で自由に変更可能**となる。
    *   *System*: 普段は「携帯サイズ (Class 1)」、使用時は「等身大 (Class 3)」、寝具としては「巨大 (Class 5)」など、用途に合わせて調整可能。

### Rule B: 欲望ムードによる補正 (By Desire Mood)
*   **【Domination (S)】**: 支配したい。 -> **Shift Down (Class 1-2)**
*   **【Submission (M)】**: 圧倒されたい。 -> **Shift Up (Class 5-6)**
