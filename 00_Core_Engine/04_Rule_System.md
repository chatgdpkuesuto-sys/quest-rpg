# GEM Fantasy RPG: Chaos Edition (Rule System)

> **「混沌こそが冒険だ。」**

本システムは `2d6` (6面ダイス2個) を使用する、シンプルかつスリリングなTRPGシステムです。

---

## 1. 判定の基本 (Core Mechanics)

全ての行動は以下の式で判定されます。

> **判定値 = `2d6 + 能力値 + 技能値` vs `目標値 (TN)`**

- **成功 (Success)**: 判定値が TN **以上**。
- **失敗 (Failure)**: 判定値が TN **未満**。
- **大成功 (Great Success)**: 判定値が TN+5 **以上**。

### 1-1. ゾロ目処理 (Critical & Fumble)
`2d6` の出目がゾロ目だった場合、以下の特殊処理が発生します。

- **成功時 (TN以上)** → **クリティカル (Critical)**
    - 効果が劇的に向上する。ダメージ2倍、追加効果など。
- **失敗時 (TN未満)** → **ファンブル (Fumble)**
    - 最悪の結果が起きる。武器破損、反撃を受けるなど。
    - **Exception**: 「6, 6」のゾロ目は常に成功扱い。「1, 1」のゾロ目は常に失敗扱い。

### 1-2. 難易度 (Target Number)
GMは以下の基準でTNを設定してください。

| TN | 難易度 | 具体例 |
|:---:|:---|:---|
| **5** | **超簡単 (Very Easy)** | 誰でもできる。壁を登る、噂を聞く。 |
| **7** | **簡単 (Easy)** | 少し練習が必要。鍵を開ける、不意打ち。 |
| **9** | **普通 (Normal)** | プロの仕事。衛兵を騙す、古文書の解読。 |
| **11** | **難しい (Hard)** | 熟練者向け。魔法の罠解除、ドラゴンに乗る。 |
| **13** | **無茶 (Extreme)** | 英雄的行為。軍隊を一人で止める。 |
| **15** | **伝説級 (Legend)** | 神業。死者蘇生、神殺し。 |

## 2. 能力値 (Ability Scores)

能力値は4つに統合され、範囲は **-1 ～ +3** です。

| 能力値 | 内容 | 主な用途 | 例 |
|:---:|:---|:---|:---|
| **STR** (筋力) | 身体的な力・持久力 | 近接攻撃、破壊、耐久判定、**HP算出** | 敵を殴る、扉を蹴破る、毒に耐える |
| **DEX** (敏捷) | 素早さ・反応速度 | 回避、射撃、隠密、精密作業 | 攻撃を避ける、鍵開け、スリ、弓攻撃 |
| **INT** (知力) | 思考力・知識量 | 魔法、調査、分析、**SP算出** | 魔法使用、魔物知識、地図解読 |
| **CHA** (魅力) | 対人能力・印象 | 交渉、説得、威圧、幸運 | 情報を聞き出す、安く買う、仲間を鼓舞 |

### ステータス算出 (Vitals)
- **HP (Hit Points)**: `5 + STR` (Min 1)
    - 耐久力。0になると戦闘不能。
- **SP (Skill Points)**: `3 + INT` (Min 1)
    - 精神力。魔法や特技の使用に消費する。足りないと使用不可。
    - **回復**: 宿屋で宿泊すると全回復。小休憩で少し回復。

---

## 3. 技能 (Skills)

特定の分野における熟練度を表します。
- **初期値**: 0
- **最大値**: +3
- **使用法**: 該当する判定時に、その技能値を加算する。

例: 「剣術 Lv2」を持つキャラが攻撃する場合 -> `2d6 + STR + 2`

---

## 4. 戦闘ルール概略
詳細は `05_Combat_Flow.md` 参照。

1.  **イニシアチブ**: `2d6 + DEX` で決定。高い順に行動。
2.  **命中判定**: `2d6 + STR/DEX + 技能` vs `敵の回避TN`。
3.  **ダメージ**: 武器の固定値 + 成功度ボーナス。
