# GEM Fantasy RPG: Chaos Edition (Rule System)

> **「混沌こそが冒険だ。」**

本システムは `2d6` (6面ダイス2個) を使用する、シンプルかつ奥深いTRPGシステムです。

---

## 1. 6つの能力値 (The 6 Stats)

キャラクターの能力は以下の6つで表されます。
範囲は通常 **0 ～ +3** です (初期作成時は+1が上限)。

| 能力値 | 英語名 | 役割・判定 | HP/SP関連 |
|:---:|:---|:---|:---|
| **Power** | Power | 近接攻撃、破壊、力仕事 | - |
| **Speed** | Speed | 回避、イニシアチブ、隠密 | - |
| **Tough** | Tough | 防御、耐久、毒耐性 | **HP算出 (5+Tough)** |
| **Mind** | Mind | 魔法攻撃、知識、意志力 | **SP算出 (3+Mind)** |
| **Charm** | Charm | 交渉、支援、幸運 | - |
| **Skill** | Skill | 器用さ、製作、解除、操作 | - |

### 算出値 (Vitals)
- **Max HP**: `5 + Tough` (初期値)
    - 耐久力。0になると気絶・命乞い状態。
- **Max SP**: `3 + Mind` (初期値)
    - スキルポイント。スキル使用時に消費する。

---

## 2. 判定の基本 (Core Mechanics)

全ての行動は以下の式で判定されます。

> **判定値 = `2d6 + 能力値 + 補正` vs `目標値 (TN)`**

- **成功**: 判定値が TN **以上**。
- **失敗**: 判定値が TN **未満**。
- **クリティカル (6,6)**: 自動成功 ＋ 効果2倍。
- **ファンブル (1,1)**: 自動失敗 ＋ アクシデント。

### 難易度 (Target Number)
| TN | 難易度 | 例 |
|:---:|:---|:---|
| **7** | 簡単 | 鍵開け、偵察、修理、鑑定、追跡 |
| **9** | 普通 | 説得、潜入、挑発、地図作成、工作 |
| **11** | 困難 | 罠設置、毒付与、不意打ち、精密射撃 |
| **13** | 極限 | 即死攻撃、範囲魔法、蘇生 |

## 3. スキル使用ルール (Skill Rules)

ユーザー定義の厳格なスキル運用ルールです。

### 3.1. 基本手順
1.  **使用宣言**: 「XXスキルを使います」と宣言。
2.  **SP消費**: コストを支払う (不足時は使用不可)。
3.  **タイプ確認**: 強化型か、判定型か、効果型かを確認。
4.  **解決**: ダイスを振る、または効果を適用する。

### 3.2. スキルタイプ
1.  **判定強化型 (Boost)**
    - *タイミング*: 判定前
    - *効果*: `2d6 + 能力値 + スキル補正`
    - 例: 「強打 (+2ダメ)」「集中 (+2魔法)」
    
2.  **判定発生型 (Check)**
    - *タイミング*: アクション時
    - *効果*: `2d6 + 指定能力値` vs `TN`
    - 例: 「解錠 (TN7)」「盗む (TN9)」

3.  **効果発動型 (Instant)**
    - *タイミング*: 即時
    - *効果*: 判定なしで効果を適用する。
    - 例: 「かばう」「暗視」

### 3.3. 回復スキル (Healing)
HPを回復するスキルは以下の固定ダイスを使用します。
対象のHPが最大値の場合、使用できません（SPの無駄遣い不可）。

| 種別 | 回復量 | 備考 |
|:---|:---|:---|
| **小回復** | `1d6` | ヒール、薬草など |
| **中回復** | `2d6` | 高級ポーション、上級魔法 |
| **大回復** | `3d6` | 奇跡、エリクサー |

- **上限**: 最大HPを超えた回復量は無効となる。

---

## 4. 戦闘ルール (Combat)

1.  **イニシアチブ**: `2d6 + Speed` で行動順を決定。
2.  **攻撃**:
    - 近接: `2d6 + Power` vs `TN (敵の回避)`
    - 射撃: `2d6 + Speed` vs `TN`
    - 魔法: `2d6 + Mind` vs `TN`
3.  **回避**: `2d6 + Speed` (リアクションスキルがない限り、通常は受動判定＝TN固定)。
    - *※PC側は「防御力(Armor)」でダメージを減らすのが基本です。*
4.  **ダメージ**: 成功時、武器/スキルのダメージボーンを適用。

---

## 5. 休息 (Rest)
- **小休憩**: 10分休息。HPを `1d6` 回復。SPは回復しない。
- **大休憩 (宿泊)**: 安全な場所で一晩。HP/SP 全回復。
