# コマンドシステム (Chat & Commands)

このゲームは、**「自然言語による宣言」**を基本とします。
従来のコマンド(`/attack`)は、GMが処理するための裏コードとして扱われます。

## 1. 基本入力 (Basic Input)
プレイヤーはチャット欄に「やりたいこと」を書くだけです。

- **攻撃**: 「剣で斬りかかる！」
- **スキル**: 「【強打】で攻撃する！」
- **探索**: 「部屋を見回す」「死体を調べる」
- **交渉**: 「『通してくれ』と頼む」「金を握らせる」

GM（システム）はこれを解釈し、適切なステータス判定を求めます。

## 2. スラッシュコマンド (Sys Commands)
デバッグや、明確に数値を指定したい場合に使用します。通常プレイでは不要です。

- `/r [式]`: ダイスを振る (`/r 1d20+5`)
- `/check [Stat]`: 指定ステータスで判定 (`/check STR`)
- `/save [Stat]`: セーヴィングスロー (`/save DEX`)

## 3. ゲーム進行 (Game Flow)
これらはチャットでもコマンドでも可能です。

| コマンド | チャット例 | 効果 |
|:---|:---|:---|
| `/rest` | 「少し休む」「休憩する」 | 小休憩 (HP小回復) |
| `/sleep` | 「今日はもう寝る」「宿に泊まる」 | 大休憩 (全回復/Save) |
| `/move [場所]` | 「ギルドへ行く」「森へ向かう」 | エリア移動 |
| `/shop` | 「店を見る」「買い物する」 | ショップ画面 |
| `/status` | 「ステータス確認」 | 自分の状態を表示 |

## 4. 特殊機能 (Special)
- `/img [プロンプト]`: 画像生成をリクエスト。
- `/debug`: GMの思考プロセス(DCなどの裏数値)を表示/非表示。
- `/save`: 手動セーブ。
- `/load`: ロード。
