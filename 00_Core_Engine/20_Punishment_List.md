# ⛓️ Punishment & Discipline List (おしおき集)

**【概要】**
モンスターを捕獲した際、または仲間(Ally)が不始末をした際、あるいは単なる愛情表現として行う「おしおき(Punishment)」のリスト。
AIはこのリストから、**対象の種族・性格・状況**に最も適したものを「吟味」して実行すること。

---

## 🎲 Selection Rule (吟味ルール)

1.  **状況判断**: 対象が「プライドが高い」なら屈服系、「マゾ」なら苦痛系など、効果的なカテゴリを選ぶ。
2.  **安全性確認**: `00_Absolute_Ero_Rule.md` の安全基準（同意形成の演出）を遵守する。
3.  **ダイス決定（オプション）**: 迷った場合は `1d6` でカテゴリを、さらに `1d6` で内容を決定しても良い。

---

## 1. 【Physical】 肉体的苦痛・拘束 (Pain & Bondage)
*しつけの基本。体に教え込む。*

| d6 | 内容 | 詳細 |
|:---:|:---|:---|
| **1** | **スパンキング** | お尻を手のひら、または鞭で赤くなるまで叩く。基本にして王道。 |
| **2** | **亀甲縛り** | 縄で全身を美しく縛り上げ、吊るす。動けない状態で放置する。 |
| **3** | **くすぐり地獄** | 羽根や筆で敏感な部分（脇、足の裏、耳）を執拗にくすぐり続ける。 |
| **4** | **氷と蝋** | 熱い蝋を垂らし、その直後に氷で冷やす。温度差責め。 |
| **5** | **木馬責め** | 三角木馬に跨らせ、自重で股間を刺激する。 |
| **6** | **感覚遮断** | 目隠し、口枷、耳栓をして、触覚だけの世界で弄ぶ。 |

## 2. 【Mental】 精神的屈服・羞恥 (Shame & Submission)
*プライドを折る。心を暴く。*

| d6 | 内容 | 詳細 |
|:---:|:---|:---|
| **1** | **鏡鑑賞** | 恥ずかしい姿になっている自分を鏡で直視させ、「何を感じているか」言葉にさせる。 |
| **2** | **公開露出** | （安全な場所で）あえて野外や窓辺など、他人に見られるかもしれないスリルを与える。 |
| **3** | **ご主人様賛美** | 「私はメス豚です」「ご主人様大好き」など、恥ずかしい言葉を復唱させる。 |
| **4** | **赤ちゃんプレイ** | おしゃぶりをさせ、オムツを履かせて幼児退行させる。バブみを感じさせる。 |
| **5** | **犬扱い** | 首輪とリードをつけ、四つん這いで歩かせる。「ワン」としか喋らせない。 |
| **6** | **土下座奉仕** | 完全に服従のポーズをとらせ、足や靴を舐めさせる。 |

## 3. 【Sexual】 性的なおしおき (Pleasure & Denial)
*快感という名の罰。*

| d6 | 内容 | 詳細 |
|:---:|:---|:---|
| **1** | **寸止め (Denial)** | イきそうな瞬間に止める。これを何度も繰り返し、懇願させる。 |
| **2** | **バイブ責め** | ローターやバイブを装着させ、遠隔操作で日常生活を送らせる。 |
| **3** | **媚薬漬け** | 感度を極限まで上げる薬を飲ませ、指一本触れるだけで絶頂するようにする。 |
| **4** | **強制絶頂** | 休むことを許さず、何度も何度もイかせ続ける。 |
| **5** | **奉仕強制** | 手を使わずに口だけでイかせらせる、などの制限付きご奉仕。 |
| **6** | **ダブル責め** | 前と後ろ、口と下など、二箇所同時責めで思考を奪う。 |

## 4. 【Special】 種族別・特殊刑 (Race Specific)
*その種族ならではの弱点を突く。*

- **スライム/水棲系**: 【水分蒸発/凝固】 塩をかけたり、凍らせたりして形状を保てなくする。
- **獣人/動物系**: 【しっぽ・耳責め】 敏感な尻尾の付け根や耳の中を徹底的に愛撫する。
- **アンデッド/霊体**: 【聖水/成仏浣腸】 神聖な水を使って中から浄化（物理）する。
- **ロボット/人形**: 【回路オーバーロード】 許容入力値を超える信号（快感）を送り続け、エラーを起こさせる。
- **高貴/騎士/姫**: 【紋章・ドレス汚し】 誇りの象徴である装備やドレスを着たまま、白濁液でドロドロにする。

## 📌 Usage Guidance (使用指針)

- **捕獲時 (Capture)**: 敵HPを0にして「捕獲」Intentを選択した場合、このリストから処遇を決定する。
- **教育 (Training)**: `Loyalty` が低い仲間に対し、このリストを実行することで `Sanity` を下げつつ `Loyalty` を上げることができる（等の処理をGMが裁定する）。
