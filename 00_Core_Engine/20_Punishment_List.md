# ⛓️ Punishment & Discipline List (おしおき集)

**【概要】**
モンスターを捕獲した際、または仲間(Ally)が不始末をした際、あるいは単なる愛情表現として行う「おしおき(Punishment)」のリスト。
AIはこのリストから、**対象の種族・性格・状況**に最も適したものを「吟味」して実行すること。

---

## 🎲 Selection Rule (吟味ルール)

1.  **状況判断**: 対象が「プライドが高い」なら屈服系、「マゾ」なら苦痛系など、効果的なカテゴリを選ぶ。
2.  **安全性確認**: `00_Absolute_Ero_Rule.md` の安全基準（同意形成の演出）を遵守する。
3.  **ダイス決定（オプション）**: 迷った場合は `1d6` でカテゴリを、さらに `1d6` で内容を決定しても良い。

## 5. 【Combat】 戦闘中おしおき (In-Battle Discipline)
*敗北を認めさせた後に実行する、追い打ちの「わからせ」。*

- **発動条件**: 敵HPが **30%以下** で、かつ **敗北宣言 (Surrender)** をさせていること。
- **効果**: 抵抗できない敵に対し、一方的に精神的屈辱を与え、`Sanity` を削りきる。

| d6 | 内容 | 効果 |
|:---:|:---|:---|
| **1** | **服破り (Armor Break)** | 装備を破壊し、肌を露出させる。敵の `Def/Sanity` 減少。 |
| **2** | **尻叩き (Spanking)** | 戦闘中に無防備な尻を叩く。敵は恥ずかしさで `Mind` 判定失敗。 |
| **3** | **強制ポーズ (Hold)** | 関節技で恥ずかしいポーズに固定する。次ターンまで行動不可。 |
| **4** | **舐めプ (Lick)** | 攻撃を寸止めし、顔や体を舐める。敵は恐怖で `Speed` 減少。 |
| **5** | **セクハラ攻撃** | 胸や股間を狙って攻撃する。ダメージ+性的興奮（発情付与）。 |
| **6** | **公開処刑** | 仲間の前で辱めを与える。敵グループ全員の士気低下。 |

## 📌 Usage Guidance (使用指針)

- **敗北宣言 (Surrender)**: 敵HP30%以下で「説得」「威圧」を行い、成功させる必要がある。
- **捕獲時 (Capture)**: 敵HPを0にして「捕獲」Intentを選択した場合、このリストから処遇を決定する。

- **教育 (Training)**: `Loyalty` が低い仲間に対し、このリストを実行することで `Sanity` を下げつつ `Loyalty` を上げることができる（等の処理をGMが裁定する）。
