# GEM Fantasy: Instruction Manual (取扱説明書)

ようこそ、剣と魔法、そして欲望の交錯する世界へ。
このファイルは、プレイヤーとGMの両方が**「迷わずに遊ぶための羅針盤」**です。

---

## 🏆 1. ゲームの目標 (Objectives)

まずは、自分がこの世界で何を成し遂げたいか決めましょう。

### A. 【ハーレム王】を目指す
世界中の美女（ヒロイン、エルフ、魔王の娘など）を仲間にし、全員の好感度をMAXにする。
*   **難易度**: 高 (資金と精神ケアが必要)
*   **クリア条件**: **[ハウジング](12_Housing_System.md)** で家を購入し、5人以上の恋人と同居する。

### B. 【世界最強】を目指す
ダンジョンの深層に眠る「神」や「システム管理者」を倒す。
*   **難易度**: 極 (戦闘ビルドの最適化が必要)
*   **クリア条件**: **ランクSクエスト**「創造主の試練」をクリアする。

### C. 【気ままな冒険者】として生きる
釣りやカジノに興じ、気の向くままに旅をする。
*   **難易度**: 低〜中
*   **クリア条件**: 特になし。あなたが飽きるまで、物語は続きます。

## 🔰 2. 厳格な冒険サイクル (Strict 5-Phase Cycle)
 
 ゲームは **Phase 1 〜 Phase 5** の手順で進行します。
 詳細は 👉 **[Master Index](00_Index.md)** を参照してください。
 
 ### Phase 1: Preparation (準備 @街)
 - **3 Action Point**: 施設利用、イベント、クエスト受領から3回選択。
 
 ### Phase 2: Adventure (冒険 @クエスト)
 - **探索ループ**: シーン進行 -> 判定 -> イベント。
 
 ### Phase 3: Combat (戦闘 @遭遇)
 - **敵との戦い**: イニシアチブ -> ターン制バトル。
 - **屈服奥義**: 敵HP30%以下で捕獲可能。
 
 ### Phase 4: Reward (報酬 @決算)
 - **戦利品**: ドロップ、報酬金、名声。
 - **成長**: **ボス撃破時のみ Stat +1**。
 
 ### Phase 5: Rest (休息 @夜)
 - **ヒロイン**: 親密度イベント。
 - **調教**: モンスター娘との戯れ。
 - **翌日へ**: 強制的にPhase 1に戻る。

## 🏠 3. 施設の機能 (Facilities Guide)

街にある施設の役割まとめ。

| 施設名 | 主な機能 | 重要性 |
|:---|:---|:---|
| **冒険者ギルド** | クエスト受注、昇格試験 | **必須**。ここがないと仕事がない。 |
| **宿屋 (Inn)** | HP/MP回復、Sanity回復 | 高い部屋ほど精神が回復する。 |
| **武器屋** | 売買、**修理**、強化 | 武器は使えば劣化する。定期的なメンテが必要。 |
| **教会** | 状態異常解除、**治療** | 敗北時の心身ケア。寄付が必要。 |
| **夜の店** | 情報収集、ギャンブル | 裏社会とのコネクションや、大人の遊び。 |

## 👑 4. GMのためのガイド (For Game Masters)

進行に迷ったら、ここをチェックしてください。

### Q1. 何をしていいか分からない！ (Stuck?)
迷ったらダイスを振って、ハプニングを起こしましょう。
*   👉 **[街イベント表](18_Town_Events.md)**: 街中でのハプニング生成。
*   👉 **[NPCジェネレーター](12_NPC_Generator.md)**: 即興で店主や通行人を作成。

### Q2. 判定の難易度(TN)はどう決める？
迷ったら **TN 9 (Normal)** を基準にしてください。
*   **7**: 簡単 / **9**: 普通 / **11**: 困難 / **13**: 極限

### Q3. NPCの反応に困った！
プレイヤーが変なことを言ってきたら、**[ソーシャルシステム](10_Social_System.md)** で対抗ロールを行います。
*   (例) 嘘を見抜く: `Insight` vs `Deception`

### Q4. 戦闘が長引いている！
無理に全滅させる必要はありません。「敵が逃走した」「命乞いをして情報を吐いた」ことにして戦闘を終了させてください。
