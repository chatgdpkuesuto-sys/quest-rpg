# ランダムイベント表 (Random Events with Luck)

**GM用ツール**: 移動中や探索中にダイス(d100)を振ってハプニングを生成する。
イベント発生時、PCは「LUCK判定」を行うことで結果を良きものに変えられる可能性がある。

---

## 1. 街道・野外 (Wilderness Travel)
街から街への移動中。危険とチャンス。

| d100 | イベント内容 (Event) | LUCK判定 (DC 12) |
|:---:|:---|:---|
| 01-10 | **平和な一日** | **成功**: Sanity+2 (良い天気)。<br>**失敗**: Sanity+1 (普通)。 |
| 11-25 | **モンスターの痕跡** | **成功**: 敵の正体と弱点が分かる。<br>**失敗**: 不意打ちを受ける(Surprise)。 |
| 26-35 | **行商人の馬車** | **成功**: 珍しい品(Bランク)を売ってくれる。<br>**失敗**: 通常品のみ。 |
| 36-45 | **謎の果実** | **成功**: 全回復する美味な果実。<br>**失敗**: 腹を壊す(毒状態)。 |
| 46-55 | **天候急変** | **成功**: 洞窟を見つけて雨宿り(全回復)。<br>**失敗**: ずぶ濡れで疲労(Exhaustion)。 |
| 56-60 | **旅の吟遊詩人** | **成功**: 感動的な歌でSanity全快。<br>**失敗**: 下手な歌で小銭をせびられる。 |
| 61-65 | **落とし穴** | **成功**: 寸前で回避。<br>**失敗**: 落下(`1d6`ダメ)し、足を挫く。 |

| 71-75 | **古い石碑** | **成功**: 古代魔法(スクロール)を発見。<br>**失敗**: 呪いを受ける(Curse)。 |

| 81-85 | **温泉発見** | **成功**: `最大HP +5` (永続)。<br>**失敗**: `HP全回復` (一時的)。 |

| 91-95 | **ラッキースケベ** | **成功**: 楽しいハプニング(好感度UP)。<br>**失敗**: 激怒される(好感度DOWN, ビンタ)。 |
| 96-99 | **流れ星** | **成功**: 願いが叶う(LUCK UP)。<br>**失敗**: 近くに落ちてくる(DEX回避)。 |
