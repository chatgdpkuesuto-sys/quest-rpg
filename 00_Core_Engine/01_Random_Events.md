# ランダムイベント表 (Random Events)

**GM用ツール**: 移動中や探索中にダイス(d100)を振ってハプニングを生成する。

## 1. 街道・野外 (Wilderness Travel)
街から街への移動中。危険とチャンス。

| d100 | イベント内容 (Event) | 効果・処理 |
|:---:|:---|:---|
| 01-10 | **何事もなく平和** | 日向ぼっこびより。全員のSanity+1。 |
| 11-30 | **モンスターの襲撃** | ランダムエンカウント(Easy)。 |
| 31-40 | **強敵の影** | 遠くに強力なモンスターが見える。隠れるか(Stealth)、挑むか。 |
| 41-50 | **行商人と遭遇** | ポーションや食料をその場で買える。 |
| 51-55 | **怪しい薬売り** | 安いが副作用のある薬を売ってくる。 |
| 56-65 | **天候悪化** | 大雨/吹雪。移動速度半減、疲労判定(CON)あり。 |
| 66-70 | **旅の吟遊詩人** | 10Gで歌を聞かせてもらう。全員に[鼓舞]バフ。 |
| 71-75 | **山賊の罠** | 落とし穴(`DEX save`)、または通行料を要求される。 |
| 76-80 | **遭難者発見** | 助けると謝礼(100G)をもらえる。見捨てるとKarma低下。 |
| 81-85 | **謎の遺跡発見** | 未発見のダンジョン入り口を見つける(要:探索)。 |
| 86-90 | **温泉発見** | 野外温泉！全員のHP/Sanity全回復。混浴イベントチャンス。 |
| 91-95 | **神の祠** | 祈るとランダムなステータスが1日+1される。 |
| 96-99 | **ラッキースケベ** | 風でヒロインのスカートがめくれる等の事故。Sanity+5, Loyalty変動なし。 |
| 00 | **竜の通過** | 上空を伝説のドラゴンが通過する。その威圧感で全員動けなくなる。 |

## 2. 街中・都市 (Urban Encounter)
滞在中や買い物中。

| d100 | イベント内容 (Event) | 効果・処理 |
|:---:|:---|:---|
| 01-20 | **平和な日常** | 特に何も起きない。 |
| 21-30 | **スリ被害** | `Perception vs DEX`。失敗すると所持金の10%を盗まれる。 |
| 31-40 | **喧嘩の仲裁** | 酔っ払いが絡んでくる。STRまたはCHAで解決。 |
| 41-50 | **迷子の子供** | 親を探してあげる。お礼に飴玉をもらう。 |
| 51-60 | **セールの開催** | 武器屋または道具屋の商品が20%オフ。 |
| 61-70 | **お祭り** | 屋台が出ている。食べ歩きデートのチャンス。 |
| 71-75 | **勧誘** | 怪しい宗教、またはギルドからの引き抜き。 |
| 76-80 | **貴族の通行** | 道を空けなければならない。態度が悪いと投獄イベント。 |
| 81-85 | **落とし物** | 財布を拾う。届ける(Karma UP)か、ネコババする(Gold UP)。 |
| 86-90 | **ナンパ撃退** | ヒロインがナンパされている。助けるとLoyalty大アップ。 |
| 91-95 | **裏路地の取引** | 闇市場への招待状を手に入れる。 |
| 96-00 | **運命の出会い** | 新しいNPC(またはヒロイン候補)とぶつかる。「食パンくわえた少女」等。 |

## 3. ダンジョン (Dungeon Exploration)
探索中の緊張感。

| d100 | イベント内容 (Event) | 効果・処理 |
|:---:|:---|:---|
| 01-15 | **敵の気配なし** | 順調に進む。 |
| 16-35 | **ワンダリング・モンスター** | 巡回中の敵と遭遇。 |
| 36-45 | **宝箱発見** | 罠があるかもしれない(`Investigation`)。中身はランダム。 |
| 46-50 | **ミミック！** | 宝箱だと思ったら敵だった。不意打ちを受ける。 |
| 51-60 | **古いキャンプ跡** | 安全地帯。小休憩(`Short Rest`)が可能。 |
| 61-65 | **冒険者の死体** | 持ち物を漁れる(Loot)。たまにゾンビ化して起き上がる。 |
| 66-70 | **謎の石碑** | 解読するとダンジョンの地図やボスの弱点がわかる。 |
| 71-75 | **落盤/崩落** | `DEX save`。失敗するとダメージ+進行不能。 |
| 76-80 | **魔法の泉** | 飲むと回復するか、毒を受けるか(50/50)。 |
| 81-85 | **行商人の亡霊** | 幽霊がアイテムを売ってくれる(通貨はSAN値)。 |
| 86-90 | **ライバルパーティ** | 他の冒険者たちと鉢合わせる。交渉か、戦闘か。 |
| 91-95 | **エッチな罠** | スライム風呂、触手穴など。ヒロインが引っかかる。 |
| 96-00 | **レアモンスター** | はぐれメタル的な希少種。倒すと大量のXP/Gold。 |
