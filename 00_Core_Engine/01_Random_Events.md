# ランダムイベント表 (Random Events with Luck)

**GM用ツール**: 移動中や探索中にダイス(d100)を振ってハプニングを生成する。
イベント発生時、PCは「LUCK判定」を行うことで結果を良きものに変えられる可能性があります。

---

## 1. 街道・野外 (Wilderness Travel)
街から街への移動中。危険とチャンス。

|:---:|:---|:---|

| 26-35 | **行商人の馬車** | **Success**: 珍しい品(Bランク)を売ってくれる。<br>**Fail**: 通常品のみ。 |
| 36-45 | **謎の果実** | **Success**: 全回復する美味な果実。<br>**Fail**: 腹を壊す(Poisoned)。 |
| 46-55 | **天候急変** | **Success**: 洞窟を見つけて雨宿り(全回復)。<br>**Fail**: ずぶ濡れで疲労(Exhaustion)。 |
| 56-60 | **旅の吟遊詩人** | **Success**: 感動的な歌でSanity全快。<br>**Fail**: 下手な歌で小銭をせびられる。 |
| 61-65 | **落とし穴** | **Success**: 寸前で回避。<br>**Fail**: 落下(`1d6`ダメ)し、足を挫く。 |
| 66-70 | **遭難者** | **Success**: 正直な商人で、街で謝礼を約束。<br>**Fail**: 盗賊の罠(Combat Start)。 |
