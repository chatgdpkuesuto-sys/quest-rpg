# 経済・ショップシステム (Economy System)

冒険に必要な装備から、裏社会で取引される「生きた商品」まで、あらゆるものが金(Gold)で解決できます。

## 1. 通貨 (Currency)
- **1 ゴールド (G)** = 100 シルバー (S) = 10,000 カッパー (C)
- **相場感**:
    - 宿屋(1泊): 1 G
    - 一般的な剣: 10-15 G
    - 奴隷1人: 500-2,000 G

---

## 2. 一般商店 (General Store)
冒険の必需品を取り扱っています。

| 商品名 | 価格 | 効果 |
| :--- | :--- | :--- |
| **松明 (Torch)** | 1 S | 1時間、周囲を照らす。 |
| **保存食 (Rations)** | 5 S | 1日分の食料。 |
| **つるはし (Pickaxe)** | 2 G | 壁を掘ることができる。要・筋力と時間。 |
| **ダイナマイト (Dynamite)** | 50 G | **壁・障害物を破壊**する。範囲20ftに3d6雷鳴ダメージ。 |
| **治癒ポーション** | 50 G | HPを `2d4+2` 回復する。 |
| **解毒剤** | 30 G | 毒状態を解除する。 |

## 4. 闇市場 (Black Market)
表通りでは買えない「違法品」や「生きた商品」を扱っています。
**入場条件**: `Thieves' Cant` を理解しているか、コネクション(200Gの賄賂)が必要。

### 特殊アイテム (Forbidden Items)
| 商品名 | 価格 | 効果 |
| :--- | :--- | :--- |
| **催眠アプリ/魔道具** | 2,000 G | 対象に使用し、WISセーヴ(DC15)に失敗すると**「洗脳(Hypnotized)」**状態にする。 |
| **隷属の首輪** | 500 G | 装着者の自由意志を奪い、命令に逆らえなくする。外すには専用の鍵が必要。 |
| **惚れ薬 (Love Potion)** | 300 G | 1時間、飲んだ相手を「魅了」状態にし、好感度を最大にする(効果終了後に戻る)。 |
| **強力な爆薬 (C4)** | 200 G | ダイナマイトの強化版。鉄の扉さえも吹き飛ばす (10d6ダメージ)。 |

### 人身売買 (Slave Trade)
捕獲されたモンスターや、わけありの少女を購入できます。
購入したキャラクターは**「所有物」**となり、パーティメンバーとして連れ歩けます。

| 商品ランク | 価格 | 内容 |
| :--- | :--- | :--- |
| **ランク C (雑用)** | 300 G | ゴブリン、コボルドなど。戦闘力は低いが荷物持ちになる。 |
| **ランク B (一般)** | 800 G | 町娘、下級戦士。容姿は整っている。 |
| **ランク A (戦闘)** | 2,000 G | エルフの魔術師、女騎士など。即戦力かつ美形。 |
| **ランク S (幻想)** | 10,000 G+ | サキュバス、姫、高位の聖女など。 |

#### 洗脳・操り人形化 (Puppet Rules)
「隷属の首輪」や「催眠アイテム」を使用されたキャラクターは、以下の状態になります。
- **自我消失**: 感情や意思を持たず、主人の命令に機械的に従う。
- **絶対服従**: 自害などの命令も拒否できない。
- **好感度固定**: 「信仰(Worship)」状態で固定され、裏切ることがなくなる。
- **デメリット**: 自発的な行動（工夫や提案）をしなくなるため、柔軟性に欠ける。
